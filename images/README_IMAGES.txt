Required Images for the Website
================================

Please add the following images to this directory:

1. greenhouse_system.png
   - Image showing the Texas A&M AgriLife Precision Automated Phenotyping Greenhouse
   - Recommended size: ~800-1000px width

2. pipeline_overview.png
   - Figure 2 from your paper: Comprehensive pipeline for multispectral plant phenotyping
   - Recommended size: Full width (100%)

3. dataset_summary.png
   - Table or visualization showing PGP v2 dataset statistics
   - Recommended size: Full width (100%)

4. segmentation_results.png
   - Figure 3 from your paper: Qualitative comparison of segmentation performance
   - Recommended size: Full width (100%)

5. feature_extraction.png
   - Figure 5 from your paper: Temporal trajectories for features
   - Recommended size: Full width (100%)

6. arxiv.jpg
   - ArXiv logo (you can download from arxiv.org or use a similar icon)
   - Recommended size: ~108px width

7. dataset.png
   - Dataset/icon image for the dataset link
   - Recommended size: ~116px width

8. code.png
   - Code/GitHub icon for the code repository link
   - Recommended size: ~116px width

9. logo.png
   - Texas A&M University or lab logo
   - Recommended size: ~170px width

10. webicon.jpg
    - Favicon for the website (small square icon)
    - Recommended size: 32x32 or 64x64 pixels

Tips:
- Extract figures from your PDF paper and save them as PNG files
- Ensure images are high resolution for web display
- You can use tools like ImageMagick or online converters to extract images from PDF
